# ①課題番号-プロダクト名
メモ帳アプリ（Localstorage活用）

## ②課題内容（どんな作品か）

子どもが病気になったとき、ママ・パパのタスクを見える化するメモアプリです。

実際に今週に娘二人が喘息になり、ママ側のタスクが多すぎてストレス発散がてら作成いたしました。作りながら、パパに「私はどれだけタスクこなしているか知っているかい？」とどや顔でタスク数の結果をみせて、ちょっとすっきりしました。

また、翌日にパパが半休をとって、私の仕事の時間をつくってくれたのは、このアプリの成果だと思います（笑）

## ⑤工夫した点・こだわった点
-はじめは、シンプルなメモ帳アプリを作成しようと思っていました。
ただ、タスクを記入しはじめたら、あまりにも娘たちの病気対応のタスクが多すぎて、可視化してみるとどうかとアイディアがうまれました。
また、最初は、単にタスクを書いていこうかと思いましたが、パパとママのどちらがタスクをこなしたかの結果がでると面白いと思い、機能を追加していきました！


## ⑥難しかった点・次回トライしたいこと(又は機能)

アプリの意図や使い方を同じサイトに表示することを試してみたかった。
そうすることで、ユーザー側がはじめて見たときも使い方がわかるなと思いました。ただ、その場合は、サイト全体のレイアウトやユーザー側の見え方も考える必要がでてくるので、そこまで手がまわりませんでした。。
